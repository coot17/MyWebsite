Core Four: LOOPS - A loop is a sequence of instruction s that is continually repeated until a certain condition is reached - Example of loop code in python are: for loop - (for x in range(5): print(x)) while loop - (count = 0 while count < 5: print(count) (count += 1) - An analogy for loops can be put into terms of a merry-go-round. Imagine the merry go round only does the loop while the ride has been turned on. This type of reliance is true for loops as well. They only loop if a certain thing has triggered them to loop. Sometime the loops are completed and other times they go on until the user tells them to stop. VARIABLES - A variable is a storage location paired with an identifier, which contains some quantity of information reffered to as a value - Example of a variable in python is: myint = 7: print(myint) - myint is now a variable. - An analogy of a variable is - when you name a pet, you keep that name stored in your brain but you can change the name of your pet at anytime and remeber that instead.
